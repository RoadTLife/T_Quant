# TA-Lib（20260601）

- 之前采用backtrader 的指标不够用，需要添加一些k线指标，所以引入了Ta-Lib，指标更加丰富，其底层用c写的速度更快。其指标有61种的k线图（可以用交易行为学用这个实现交易）
- ta-lib就是简单的numpy数组，基础用法
- 止损ATR（在海归中用的比较多）
- RSI 进入10-20 % 会钝化，可能会接刀，所以可以等其从10-20向30的时候配合量能入场，减少次数增加机率
- ATR  计算当前最高价-最低价 ，还考虑昨天收盘和今天开盘的跳空缺口，ATR就是TR的N日移动平均，其值飙升表示市场极度疯狂或者恐慌，ATR极低表示死水区，他告诉的是振幅，止损为入场价-2\*ATR
- 市场处于那种趋势，怎么判断市场行情ADX
