## 六、情绪综合评分

### 评分模型
| 维度 | 权重 | 指标 |
|------|------|------|
| 涨跌停情绪 | 30% | 涨停数、跌停数、连板数、封板率 |
| 市场广度 | 25% | 涨跌比、上涨比例、量能 |
| 指数表现 | 20% | 主要指数涨跌幅 |
| 资金流向 | 15% | 北向资金、两融余额 |
| 海外影响 | 10% | A50期货、美股表现、汇率 |

### 情绪等级划分
| 评分区间 | 等级 | 操作建议 |
|----------|------|----------|
| 80-100 | 极度亢奋 | 谨慎追高，注意回调风险 |
| 60-80 | 偏乐观 | 可适度参与强势板块 |
| 40-60 | 中性 | 观望为主，等待明确信号 |
| 20-40 | 偏悲观 | 控制仓位，防御为主 |
| 0-20 | 极度恐慌 | 关注超跌机会，分批低吸 |

## 七、数据采集任务配置

### 采集时间窗口
- **开始时间**: 09:15（集合竞价开始）
- **结束时间**: 09:30（开盘）
- **数据刷新**: 每分钟更新一次

### 数据存储
- 表名: `trade_market_sentiment`
- 存储频率: 每日开盘前快照
- 保留周期: 永久保留

### 告警阈值
| 指标 | 阈值 | 告警级别 |
|------|------|----------|
| 涨停数 > 100 | 高情绪 | 提示 |
| 跌停数 > 50 | 恐慌 | 警告 |
| 涨跌比 < 0.5 | 弱势 | 警告 |
| 炸板率 > 30% | 分歧 | 提示 |

## 八、情绪报告输出

### 报告结构
1. **核心情绪指标**（涨跌停、涨跌比、封板率）
2. **指数表现概览**（各主要指数情况）
3. **热点板块分析**（领涨领跌板块）
4. **海外市场影响**（A50、美股、汇率）
5. **消息面汇总**（重要新闻和公告）
6. **综合情绪评分**（0-100分）
7. **操作建议**（基于情绪等级）

### 输出格式
- 文本报告（用于日志和通知）
- JSON数据（用于系统对接）
- 可视化图表（用于Dashboard）
