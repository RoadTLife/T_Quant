# 缠论

- 中枢和笔（感觉这个老师有点差，不是很强，chan 这个项目的效果比较好），线段描述的是方向，中枢描述的是犹豫，缠论不是严格上的定义
- 通过缠论确定买卖点
  - 一买：趋势衰竭（中枢下方），抄底
  - 二买：确定反转（中枢附近），确定
  - 三买：中枢突破（上分），追涨
- 中枢A 和 中枢B， 绿柱是否萎缩（背驰），C的MACD绝对面积< B的MACD面积\*80%
- 缠论不预测方向，但识别市场的动能翻转
- 测试动态止盈（这个可以增加胜率，后面自己可以采用，但是在趋势强的时候会被摔下车） **所以需要针对不同股票采用不同策略**， 先选出股，在选择策略
- 高波动+有趋势=》 增加策略（阶段止损+ATR跟踪）
- 低波动+强趋势=》基础策略+更宽止盈
- 高波动+无趋势=》不交易（等三买信号）
- 各个组合可以结合起来使用
- 交易波动大的一定要学会止损和保护利润

总结：

- 选股：基本面+板块轮定+概念热点
- 择时：缠论（结构信号）+海龟（趋势跟踪）+TA-lib（技术指标）
- 风控：ATR仓位管理 + 缠论止损 + 多周期过滤
- 增强：LiggtGBM 信号过滤+特征工程
